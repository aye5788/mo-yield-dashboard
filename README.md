# MO Yield Regime Dashboard

Streamlit dashboard that applies an empirical dividend-yield regime model to Altria (MO).

## Features
- Live MO price (Alpha Vantage)
- Live dividend data
- Yield regime classification
- Forward return expectation
- Buy-zone price ladder

## Deployment (Streamlit Cloud)
1. Push repo to GitHub
2. Create Streamlit Cloud app
3. Add secret:
   ALPHA_VANTAGE_KEY = "your_api_key"
4. Deploy
